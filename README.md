# React Router Assignment
This project uses react-router-dom for navigation between different pages.
Pages included:
* Home
* Dashboard
* Login
* Signup

I used BrowserRouter, Routes and Route components to create different routes. Navigation between pages is done using Link.

## How to run
Install dependencies
```bash
npm install
```

Install react-router-dom
```bash
npm install react-router-dom
```

Start the project
```bash
npm run dev
```

Open the localhost link shown in the terminal.

## What I learned
* BrowserRouter
* Routes and Route
* Link component
* Navigation between pages without reloading the page
